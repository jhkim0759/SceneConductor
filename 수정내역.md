## 🐞 버그 수정 (신규 클론에서 실패하던 원인)

- **GALP `run_galp.py`** — `TypeError: got multiple values for argument 'device'` 해결.
  서브모듈 GALP의 `init_ss_generator_v1_4` 시그니처가 바뀐 것이 원인.
  `inspect.signature`로 분기하여 **vendored / submodule 두 버전 모두** 동작하게 수정.
- **GroundingDINO `_C` 빌드 (`setup.sh`)** — `NameError: _C is not defined` 해결.
  torch 2.9.1과 안 맞는 `ms_deform_attn_cuda.cu`를 빌드 전에 자동 패치
  (`.type().is_cuda()`→`.is_cuda()`, `AT_DISPATCH ... x.type()`→`x.scalar_type()`),
  빌드 후 `groundingdino._C` import를 **검증**(실패 시 조용히 넘어가지 않고 FAILED 표시).

## ✨ 신규 스킬: `sceneconductor-setup`

클론 한 번으로 전체 세팅을 끝내는 스킬 (`/sceneconductor-setup`).
서브모듈 → Blender → conda 5개 환경 → 체크포인트 → 검증을 한 번에 수행.

- `SKILL.md` — 트리거/사용법 정의
- `scripts/provision.sh` — 5단계 마스터 오케스트레이터 (각 단계 생략·재실행 가능)
- `scripts/download_checkpoints.sh` — GroundedSAM / GALP / SAM3D(게이트) / Qwen 가중치 다운로드 + SAM3D 심볼릭링크
- `scripts/verify_install.sh` — Blender·환경·`_C`·체크포인트·서브모듈 PASS/FAIL 점검 (읽기 전용)
